本以为这些li，会分为两排，但是，第二组中的第1个li，去贴靠第一组中的最后一个li了。这是为什么呢？

1. 因为浮动的孩子无法为父容器撑起高度，同时浮动的孩子本身会失去默认的宽高

2. 浮动后的元素有了贴边的效果，如果是左浮动，那么向左边贴了

3. 可以看到第二排的列表比第一排的列表稍微偏下了一点点，那是因为给ul设置了border，所以两个ul的高度为8px，第二个ul按照标准流的布局，所以在第一个ul的下面

4. 如果不给ul设置border，那么两个列表的高度就是一样的，因为此时div和ul的高度都为0
