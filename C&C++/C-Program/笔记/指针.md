# 指针

## 1 指针概念

指针：一个变量的(内存)地址称为该变量的“指针”，通过指针能找到以它为地址的内存单元。
指针概念是构成C/C++的重要元素之一，是变量的一种类型，存放的是指定类型数据的地址，而同类型变量存放的是具体的数据。

指针变量：指针变量是用来存放另一个变量的地址的变量。指针是概念，指针变量是具体实现。

每一个变量都有一个内存位置，每一个内存位置都定义了可使用连字号（&）运算符访问的地址，它表示了在内存中的一个地址。
