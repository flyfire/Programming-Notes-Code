# C标准库

---
## C语言标注库包括库

| 文件 | 简介说明 |
|---|---|
| `<assert.h>` | 断言相关 |
|`<ctype.h>` | 字符类型判断 |
| `<errno.h>` | 标准报错机制 |
| `<float.h>` | 浮点运算 |
| `<limits.h>` | 各种体系结构限制 |
| `<locale.h>` | 本地化接口 |
| `<math.h>` | 数学函数 |
| `<setjmp.h>` | 跨函数跳转 |
| `<signal.h>` | 信号|
| `<stdarg.h>` | 可变参处理 |
| `<stddef.h>` | 一些标准宏定义 |
| `<stdio.h> `| 标准I/O库 |
| `<stdlib.h>` | 标准工具库函数 |
| `<string.h>` | 字符串处理函数 |
| `<time.h>`| 时间相关 |
| `<stdint.h>和<inttypes.h>`| C99增加的可移植类型 |
