## 3 Range

range 是对List的一种扩展，变量定义和大体的使用方法如下：

        def aRange  = 1..3   // range类型有begin值 加两个点 +end 值表示  左边这aRange包含 1 , 2 , 3 .这三个值
        
        println("aRange = " + aRange)
        
        //如果 不想包含最后一个元素，则可以使用<，
        def aRange2 = 1..<10
        println("aRange2 = " + aRange2)
        
        println(aRange.from)
        println(aRange.getFrom())
        println(aRange.to)
